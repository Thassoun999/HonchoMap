# Honcho Campout 3D Map

## Project Details

The following is a 3D Recreation of the Honcho Campout campgrounds. All assets (except for the ones list below) are made by Sydra (Tae) Hassoun in Blender 4.5.2
The project is made using Three.JS, react-three-fiber, as well as react-drei one a Vite framework.

Display and usage of the Honcho Campout 3D Map is commercially reserved for Honcho and Honcho Campout and is to only be displayed on the Honcho Campout Website.
(https://www.honchocampout.com/)
